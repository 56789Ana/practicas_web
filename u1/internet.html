¿Qué es internet?

red de computadoras interconectadas entre sí a nivel mundial con el objetivo de hacer común información de acceso público. Esta red utiliza un lenguaje en común para la comunicación entre los dispositivos

¿Qué es un proveedor de internet?

Un proveedor de servicios de Internet (ISP) es una empresa u organización que les proporciona a las personas, las empresas u otras organizaciones el acceso a Internet, por medio de una variedad de tecnologías como banda ancha, DSL, cable, fibra óptica, satélite y conexiones inalámbricas.

Carpeta local vs carpeta remota

  • Carpeta local: Se utiliza para almacenar datos que son específicos de un sistema de Windows, lo que significa que los datos no están sincronizados entre PCs.
  • Carpeta remota: Es la carpeta donde se almacenan los archivos de un sitio en el servidor web.

¿Que es la experiencia de usuario?

Proceso afectivo interno de que resulta la interacción que un usuario tiene con un producto o servicio, el cual puede ser agradable o desagradable.

Trabajar en la experiencia del usuario significa hacer que sus productos o servicios sean lo más fáciles de usar posible para captar su atención y producir en ellos una experiencia positiva.

En pocas palabras, es una medida de la satisfacción del cliente cuando se involucran con tus productos o servicios, ya sea en una tienda en línea o física.

¿Qué es la interfáz de usuario?

Concepto que abarca arquitectura de información, patrones y diferentes elementos visuales que nos permiten interactuar de forma eficaz con sistemas operativos y softwares de diversos dispositivos.
